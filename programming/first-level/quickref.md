```js
  some.code()
})
```


### Show headers

| Prefix | Example               | What     |
|--------|-----------------------|----------|
| `//`   | `//hr[@class='edge']` | Anywhere |

| `/`    | `/html/body/div`      | Root     |
{.show-header}

Add `{.show-header}` to add headers.
